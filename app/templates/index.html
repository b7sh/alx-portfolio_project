About the Project

This project was inspired by a personal challenge I faced with managing my finances. After struggling to track my income and expenses efficiently, I realized I needed a tool that could help me better understand my financial health. During my time at ALX Software Engineering, I decided to turn this personal need into a portfolio project.

This is a Portfolio Project for ALX Software Engineering. You can learn more about the ALX program here.
